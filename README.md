# 🚀 CommodaFlow

**CommodaFlow** es una plataforma profesional de gestión de tareas diseñada para optimizar el flujo de trabajo diario. Este proyecto combina un frontend robusto en **React + TypeScript** con una arquitectura de backend escalable, enfocándose en el tipado fuerte y la mantenibilidad.

## 🛠️ Stack Tecnológico

### Frontend
- **React 18** + **Vite**: Para una experiencia de desarrollo ágil.
- **TypeScript**: Garantizando la integridad de los datos en toda la aplicación.
- **Tailwind CSS**: Estilizado moderno y utilitario.
- **React Router**: Navegación fluida entre vistas (SPA).

### Backend (Próximamente)
- **Node.js** + **Express**: API RESTful.
- **Vercel**: Infraestructura de despliegue y Serverless Functions.

## 📂 Estructura del Proyecto

El proyecto sigue una organización modular para facilitar su escalabilidad:

- `src/components/`: Componentes reutilizables y atómicos.
- `src/pages/`: Vistas principales de la aplicación.
- `src/hooks/`: Lógica de estado y efectos personalizada.
- `src/types/`: Definiciones globales de interfaces y tipos.
- `server/`: Lógica de servidor y rutas de API.
- `docs/`: Documentación técnica y de gestión.
